# FutureFrom 网址导航

> 基于 [WebStackPage](https://github.com/WebStackPage/WebStackPage.github.io) 二次开发的静态网址导航网站。

## 🌟 项目介绍

FutureFrom 是一个专注于 AI 工具、科研资源、设计素材和开发者工具的导航网站。本项目基于 WebStack 开源项目进行定制和美化，旨在为用户提供更加现代、简洁且高效的资源导航体验。

## ✨ 主要特性

- **UI 全面升级**：
  - 采用现代化的深蓝/蓝紫色调配色方案
  - 重新设计的卡片样式，统一高度，视觉更整齐
  - 优化的侧边栏和导航交互，支持折叠/展开
  - 更加细腻的阴影和圆角处理
- **内容丰富**：
  - 收录了主流的 AI 模型（ChatGPT, Claude, Gemini, DeepSeek 等）
  - 聚合了科研工具、学术搜索、电子书资源
  - 提供了开发者常用的技术社区、API 平台、开源镜像站
  - 包含设计素材、影视资源等娱乐生活内容
- **响应式设计**：完美适配 PC 端和移动端访问

## 🛠️ 修改记录

相比于原版 WebStackPage，本项目进行了以下主要修改：

1.  **界面美化**：
    - 新增 `assets/css/modern.css`，覆盖了默认样式。
    - 调整了卡片 (`.box2`) 的高度、边距和悬停效果。
    - 优化了侧边栏 (`.sidebar-menu`) 的配色和选中状态。
2.  **内容更新**：
    - 移除了部分失效链接。
    - 新增了大量 AI 相关工具和科研资源站点。
    - 补全了所有站点的描述信息。
3.  **功能修复**：
    - 修复了侧边栏在折叠状态下的显示问题。
    - 修复了移动端下的部分显示异常。
